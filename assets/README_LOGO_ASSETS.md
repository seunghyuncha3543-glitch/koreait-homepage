# KOREA IT 로고 자산 - Brand Book PDF 추출본

이 폴더의 파일은 `KOREA_IT_Brand_Book_v2_4.pdf` 9페이지의 로고 시스템에서 추출한 웹용 자산입니다.

## 포함 파일

- `logo-horizontal.svg` : 홈페이지 헤더, 이메일 서명, 가로형 매체용
- `logo-stacked.svg` : 메인 Hero, 표지, 중앙 배치용
- `symbol.svg` : 파비콘, 모바일 아이콘, SNS 프로필용
- `wordmark.svg` : 제한된 가로 영역에서 사용하는 워드마크
- `favicon.svg` : `symbol.svg`와 동일한 파비콘용 별칭
- `*.png` : SVG 사용이 어려운 환경을 위한 투명 PNG

## Codex 적용 지시

1. 기존 텍스트 로고 또는 임시 로고는 모두 제거합니다.
2. 헤더에는 `assets/logo-horizontal.svg`를 사용합니다.
3. 첫 화면 중앙 로고가 필요하면 `assets/logo-stacked.svg`를 사용합니다.
4. 파비콘에는 `assets/favicon.svg` 또는 `assets/symbol.svg`를 사용합니다.
5. CSS에서 로고의 `width`만 조정하고 `height: auto`로 비율을 유지합니다.
6. 로고에 그림자, 외곽선, 색상 변경, 회전, 왜곡 효과를 적용하지 않습니다.

## 주의

이 파일은 PDF에서 추출한 웹용 자산입니다. 디자이너가 보유한 원본 AI/EPS/SVG가 있다면 그 원본이 최우선입니다.
